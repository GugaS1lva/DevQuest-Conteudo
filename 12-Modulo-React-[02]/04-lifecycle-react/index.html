Lifecycle ou Ciclo de Vida de um Componente React

Documentation

O Componente React vai ter 4 fases:

  • Inicialização do Componente:
    Na fase de Inicialização, como o nome já diz, o Componente React se prepara pra sua Inicialização. Geralmente aqui ele vai configurar os estados iniciais, usando o objeto STATE ou PROPS-PADRÕES, se houverem.
  • Montagem do Componente:
    Aqui ele está pronto pra ser montado no DOM do navegador. Nessa fase o componente vai seguir o seguinte fluxo de métodos:
    • constructor()
    • getDerivedStateFromProps()
    • render()
    • componentDidMount()
  • Atualização do Componente:
    Essa fase começa quando o Componente React já nasceu no navegador e cresce recebendo novas atualizações. Os métodos que são chamados nessa fase são:
    • getDerivedStateFromProps()
    • shouldComponentUpdate()
    • render()
    • getSnapshotBeforeUpdate()
    • componentDidUpdate()
  • Desmontagem do Componente:
    Nessa fase, o componente não é mais necessário e vai ser desmontado/retirado do DOM. O método que se chama nessa fase é:
    • componentWillUnmount()
