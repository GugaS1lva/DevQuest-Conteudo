Hooks | Ganchos

Documentation

● Hooks nos permite trabalhar com State e com LifeCycle sem usar componentes de classe.

● Hooks são funcções que permitem você a você "ligar-se" aos recursos de State e LifeCycle à partir de componentes funcinais.

● É importante saber que os Hooks NÃO FUNCIONAM DENTRO DE CLASSES

● É como se o React ficasse ouvindo modificações no State e executasse alguma ação (geralmente chamar alguma função ou executar algum trecho de código) quando o estado mudasse.

Os Dois tipos de Hooks mais importantes:

Hook de Estado - useState()

useState feito com Classe vs feito com Função:



Hook de Efeito - useEffect()

o Hook de efeito te permite executar efeitos colaterais em componentes funcionais. A gente usa esse Hook pra trabalhar com o LifeCycle, pra isso a gente vai usar o método useEffect().
Ele vai ser chamado sempre que o componente for montado e/ou atualizado.

REGRAS:

  • Não chame Hooks dentro de Loops, Condições ou Funções Aninhadas.
  • Apenas chame Hooks de Componentes Funcinais. Não chame Hooks de Funções JS comuns.
