Operadores Lógicos

&&

Se tiver qualquer valor como sendo falso dentro da minha expressão, vai SEMPRE retornar falso.

"Se algo é verdadeiro E algo é verdadeiro" = TRUE

"Se algo é falso E algo é verdadeiro" = FALSE

"Se algo é verdadeiro E algo é falso = FALSE

"Se algo é falso E algo é falso" = FALSE

||

Só vai retornar false se TODAS as operações forem falsas. Se não, retorna verdadeiro.

"Se algo é verdadeiro OU algo é verdadeiro" = TRUE

"Se algo é falso OU algo é verdadeiro" = TRUE

"Se algo é verdadeiro OU algo é falso = TRUE

"Se algo é falso OU algo é falso" = FALSE

NOT

Vai inverter o resultado final da expressão. Se for true vira false e vice versa.

let a = 4
let b = 4

//SEM o Not
console.log(a === b) -> true


//COM o Not console.log(!(a === b)) -> false
