Tags Semânticas - Parte 01


< header >

Conteúdo do cabeçalho. Pode ser usado no topo de qualquer tag, simulando o cabeçalho do conteúdo. Ex: dentro da tag body; main; article; aside.


< main >

Conteúdo principal de um site. Geralmente cada página contém apenas 1 main por questões de semântica.


< footer >

Conteúdo final de um site. Rodapé da página. Contém os créditos dos desenvolvedores, número para contato e semelhantes.


< nav >

Conteúdo de navegação. Menu de links de navegação do site. Vários links para próximas páginas ou para seções específicas do meu site.


< section >

Conteúdo de seção. Cria uma seção para englobar todos os artigos e imagens. OBS: Article são para conteúdos isolados, sections são para a junção de artigos/imagens.
