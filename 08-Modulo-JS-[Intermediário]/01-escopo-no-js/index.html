ESSE EXERCÍCIO DEVE SER ESTUDADO DENTRO DO VS CODE PARA MELHOR DESEMPENHO. Escopo no JavaScript [A acessibilidade das variáveis é limitada ao escopo onde as mesmas são criadas.]
Tipos de escopo que existe no JS: • Escopo de BLOCO: Um bloco de código no js define um escopo pra variáveis criadas com o CONST e LET. • Escopo AUTÔNOMO: Exatamente igual ao 'Escopo de bloco:'. • Escopo de FUNÇÃO: Uma função JS define um escopo para variáveis criadas com o LET, CONST e VAR. • Escopo de GLOBAL: Pode ser declarada num 'script local' dentro da página HTML e servirá pra todos os arquivos JS separados. OBS: Cuidado com o EFEITO CASCATA (ordem de execução de cada tag script)
