Flexbox - Introdução

Flexbox vai facilitar a criação de uma estrutura de layouts sem precisar usar o float: ou o position:, como posicionamento manual.
O Flexbox é mais indicado para layouts de pequena escala e componetes de um site, enquanto o Grid é mais eficiente para layouts de grande escala como uma página.




Eixo Principal => flex-direction: row(linha) | row-reverse(linha ao contrário) | column(coluna) | column-reverse(coluna ao contrário)

Se o Eixo Principal for definido como linha, o Eixo Transversal fica como coluna.
Se o Eixo Principal for definido como coluna, o Eixo Transversal fica como linha.
