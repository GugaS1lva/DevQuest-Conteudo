JSON

JSON => JavaScript Object Notation

Um Arquivo que contém uma série de dados estruturados em forma de texto e é utilizado para transferir/trafegar dados entre aplicações.



JSON vs XML

JSON é mais legível, mais organizado, menos verboso.

Anatomia do JSON:

        
            {
                "chave": 'valor',
                "chave2": 'valor2'
            }
