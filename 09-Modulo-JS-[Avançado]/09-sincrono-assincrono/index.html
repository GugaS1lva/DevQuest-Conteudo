Síncrono • Assíncrono

Síncrono:

Quando a gente diz que a execução de um código é Síncrona, basicamente estamos dizendo que ela segue um fluxo linear de execução (de cima para baixo).

Só executa o bloco de código abaixo se o de cima ja tiver sido executado. basicamente uma dependência.

Assíncrono:

Quando a gente tem essa execução Assíncrona, é como se ela se descolasse do fluxo e perde a dependência, sendo executada quando ouver necessidade.
